REST Home Page

Kaikki Kysymykset

Method: GET Endpoint: /kysymykset

JSON Esimerkki: 

[
	{
	"kysymys": "Sukupuoli:",
	"kysymysid": 1
	},
	{
	"kysymys": "Status:",
	"kysymysid": 2
	}
]

Kysymys Id :llä

GET /kysymys/{id} /kysymys/2

JSON Esimerkki: 

	{
	"kysymys": "Status:",
	"kysymysid": 2
	}

Uusi Kysymys

Method: POST /kysymykset (Kokeile Postmänillä/jne.)

JSON Esimerkki: 

	{
	"kysymys": "Lempivärisi:",
	"kysymysid": 6
	}

Kaikki Vastaukset

Method: GET Endpoint: /vastaukset

JSON Esimerkki: 

[
	{
	"vastausid": 1,
	"vastaus": "Mies"
	},
	{
	"vastausid": 2,
	"vastaus": "Opettaja"
	}
]

Vastaus Id :llä

GET /vastaus/{id} /vastaus/2

JSON Esimerkki: 

	{
	"vastausid": 2,
	"vastaus": "Opiskelija"
	}

Uusi Vastaus

Method: POST /vastaukset (Kokeile Postmänillä/jne.)

JSON Esimerkki: 

	{
	"vastaus": "Muu",
    	"kysymys": {
    		"kysymysid": 1
    		}
	}
